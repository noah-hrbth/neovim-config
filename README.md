## Important steps (on Mac)

1. `rm -rf ~/.local/state/nvim/`
2. `rm -rf ~/.local/share/nvim/`
3. `rm -rf ~/.config/nvim/`
4. `git clone https://github.com/noah-hrbth/neovim-config.git ~/.config/nvim`
5. `cd ~/.config/nvim/`
6. run `nvim`

## Important commands

- `:ASToggle` turn off autosave
- `:Emmet` turn on emmet
